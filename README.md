# Chartty - Free Pie Chart Maker

A free, customizable pie chart maker. Create professional pie charts in seconds.

**Website:** https://www.chartty.io

## Features

- Multiple chart styles and color palettes
- Donut charts with adjustable inner radius
- Customizable labels, legends, and backgrounds
- Export as PNG or SVG
- Embeddable charts with shareable links
- Responsive design

## Development

```bash
npm install
npm run dev
```

## Tech Stack

- React + TypeScript
- Vite
- Zustand (state management)
- Canvas API (chart rendering)
